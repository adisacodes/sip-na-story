# Sip na Story
A two-page cocktail bar website built with Flask and Bootstrap, featuring a warm style design and a menu of classic cocktails.

## About
Sip and Story is a simple, friendly cocktail website - a homepage that welcomes visitors, and a menu page showcasing classic cocktails. The project demonstrates basic flask routing and templating combined with bootstrap for responsivness and clean styling.

## Tech Stack
- Flask
- HTML, Bootstrap, custom CSS
- Playfair Display(headings), Poppins(body text)

## Prerequisites
- Python3 installed in your machine

## Setup
clone the repository

## Create and activate a virtual environment
- python3 -m venv venv
- source venv/bin/activate

## install flask
- pip install flask

## Run the app
- python3 app.py

## Open your browser and go to:
- http://127.0.0.1:5000/
